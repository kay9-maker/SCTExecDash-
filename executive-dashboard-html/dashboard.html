Update Milestone Data

How to Update Milestone Data:

  1. Export your Confluence page to Word
  2. Open the Word document and select the entire Milestone Overview table
  3. Copy the table (Ctrl+C or Cmd+C)
  4. Paste it in the textarea below
  5. Click "Preview" to verify the data looks correct
  6. Click "Apply Changes" to update the dashboard
